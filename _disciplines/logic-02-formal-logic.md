# 形式逻辑 (Formal Logic)

> "因为房屋都必有人建造，但建造万物的就是神。" —— 希伯来书 3:4

## 引言

形式逻辑是逻辑学中最具系统性和严密性的分支，它研究推理的形式结构而不关注具体内容。正如一座建筑物必须遵循力学定律，无论其外观如何；有效的论证也必须遵循逻辑定律，无论其讨论的主题为何。

从基督教的视角看，形式逻辑的存在彰显了上帝创造的理性秩序。上帝不仅创造了物质世界的规律（如物理定律），也创造了思维世界的规律（如逻辑定律）。这两者都源于祂不变的本性，都是祂智慧的彰显。

## 命题逻辑 (Propositional Logic)

### 基本概念

命题逻辑是形式逻辑的基础，它研究由简单命题通过逻辑联结词组成的复合命题。

**命题（Proposition）**：可以判断真假的陈述句。
- 例："耶稣是基督"（真命题）
- 例："2 + 2 = 5"（假命题）
- 反例："你好吗？"（疑问句，不是命题）

### 逻辑联结词

1. **否定（Negation）**：¬P（非 P）
   - 例：P = "下雨"，¬P = "不下雨"

2. **合取（Conjunction）**：P ∧ Q（P 且 Q）
   - 只有当 P 和 Q 都为真时，P ∧ Q 才为真
   - 例："神是慈爱的" ∧ "神是公义的"

3. **析取（Disjunction）**：P ∨ Q（P 或 Q）
   - 只要 P 或 Q 有一个为真，P ∨ Q 就为真
   - 例："基督是神" ∨ "基督是人"（实际上两者都为真！）

4. **蕴含（Implication）**：P → Q（如果 P，则 Q）
   - 当 P 为真而 Q 为假时，P → Q 为假；其他情况都为真
   - 例："若承认基督为主 → 必然得救"（罗马书 10:9）

5. **等价（Equivalence）**：P ↔ Q（P 当且仅当 Q）
   - P 和 Q 同真或同假时，P ↔ Q 为真
   - 例："称义" ↔ "被算为义"

### 真值表（Truth Tables）

真值表是分析复合命题的有力工具：

**否定（¬P）**：
| P | ¬P |
|---|-----|
| T | F   |
| F | T   |

**合取（P ∧ Q）**：
| P | Q | P ∧ Q |
|---|---|-------|
| T | T | T     |
| T | F | F     |
| F | T | F     |
| F | F | F     |

**蕴含（P → Q）**：
| P | Q | P → Q |
|---|---|-------|
| T | T | T     |
| T | F | F     |
| F | T | T     |
| F | F | T     |

### 重言式（Tautologies）

重言式是永远为真的命题，例如：
- **排中律**：P ∨ ¬P（P 或非 P）
- **矛盾律**：¬(P ∧ ¬P)（不可能 P 既真又假）
- **蕴含传递性**：((P → Q) ∧ (Q → R)) → (P → R)

这些逻辑真理不依赖于经验，而是基于逻辑系统本身的结构。从神学角度看，它们反映了上帝思想的必然性——上帝不能自相矛盾。

## 谓词逻辑 (Predicate Logic)

### 超越命题逻辑

命题逻辑将整个句子视为不可分割的单位，但许多推理需要分析句子的内部结构。谓词逻辑引入了以下概念：

1. **个体（Individuals）**：特定的对象
   - 例：亚伯拉罕、以撒、雅各

2. **谓词（Predicates）**：描述个体性质或关系的表达式
   - 例：M(x) = "x 是人"
   - 例：L(x, y) = "x 爱 y"

3. **量词（Quantifiers）**：
   - **全称量词（Universal Quantifier）**：∀x（对所有 x）
   - **存在量词（Existential Quantifier）**：∃x（存在 x）

### 谓词逻辑示例

**经典三段论用谓词逻辑表达**：

前提1：∀x (M(x) → F(x))（所有人都会犯罪）
前提2：M(苏格拉底)（苏格拉底是人）
结论：F(苏格拉底)（苏格拉底会犯罪）

**圣经教义的形式化**：

1. "神爱世人"（约翰福音 3:16）
   - ∀x (P(x) → L(神, x))
   - （对所有人 x，上帝爱 x）

2. "罪的工价乃是死"（罗马书 6:23）
   - ∀x (S(x) → D(x))
   - （对所有犯罪的 x，x 必定死）

3. "唯有基督在我们还作罪人的时候为我们死"（罗马书 5:8）
   - ∃x (C(x) ∧ ∀y(S(y) → D(x, y)))
   - （存在一位基督 x，为所有罪人 y 而死）

## 推理规则

### 基本推理规则

1. **肯定前件（Modus Ponens）**：
   - 前提1：P → Q
   - 前提2：P
   - 结论：Q
   - 例：若信就得救；彼得信；故彼得得救

2. **否定后件（Modus Tollens）**：
   - 前提1：P → Q
   - 前提2：¬Q
   - 结论：¬P
   - 例：若是假先知必不应验；以赛亚的预言应验了；故以赛亚不是假先知

3. **假言三段论（Hypothetical Syllogism）**：
   - 前提1：P → Q
   - 前提2：Q → R
   - 结论：P → R
   - 例：信心 → 顺服；顺服 → 成圣；故信心 → 成圣

4. **析取三段论（Disjunctive Syllogism）**：
   - 前提1：P ∨ Q
   - 前提2：¬P
   - 结论：Q
   - 例：耶稣要么是骗子，要么是疯子，要么是主；祂不是骗子也不是疯子；故祂是主

### 量词推理

1. **全称实例化（Universal Instantiation）**：
   - ∀x P(x) ⊢ P(a)
   - 若对所有 x，P(x) 成立，则对特定个体 a，P(a) 也成立
   - 例：所有人都需要救恩 → 我需要救恩

2. **存在概括（Existential Generalization）**：
   - P(a) ⊢ ∃x P(x)
   - 若对特定个体 a，P(a) 成立，则存在 x 使得 P(x) 成立
   - 例：耶稣复活了 → 存在从死里复活的人

## 形式系统的性质

### 一致性（Consistency）

一个形式系统是一致的，如果不能从它推导出矛盾（P ∧ ¬P）。

**神学意义**：圣经的一致性反映了上帝的信实。虽然圣经由不同作者在不同时期写成，但因为有同一位圣灵的默示，它形成了一致的神学体系。

### 完备性（Completeness）

一个形式系统是完备的，如果所有在语义上为真的语句都可以在系统内被证明。

**神学反思**：人类的知识系统永远不可能完备，因为我们是有限的受造物。只有上帝拥有完全的知识（全知）。

### 可判定性（Decidability）

一个系统是可判定的，如果存在算法可以判定任何给定命题的真假。

**神学意义**：哥德尔不完备定理表明，足够复杂的形式系统不可能既完备又可判定。这提醒我们：人类理性有其局限，我们需要上帝的启示。

## 创造-堕落-救赎-成全的视角

### 创造：逻辑的根基

上帝创造万有时赋予了宇宙理性的结构。形式逻辑的规律不是人发明的，而是人发现的——它们植根于上帝的本性：

- 上帝的**不变性**保证了逻辑定律的普遍性
- 上帝的**自洽性**保证了逻辑系统的一致性
- 上帝的**全知**是一切真理的终极源泉

### 堕落：逻辑的滥用

罪人虽然不能改变逻辑规律本身，却会滥用逻辑：

1. **诡辩**：用貌似合理的论证为罪辩护
2. **骄傲**：以为人的理性可以独立于上帝判断一切
3. **虚妄**：建立貌似严密但前提错误的思想体系

例如：新无神论者使用形式逻辑论证上帝不存在，但他们的前提本身已预设了自然主义世界观。

### 救赎：逻辑的正用

基督徒应当正确使用逻辑：

1. **护教**：用严密的论证为信仰辩护（彼得前书 3:15）
2. **分辨**：用逻辑分析检验教导的一致性（使徒行传 17:11）
3. **建造**：用清晰的论证建立神学体系

奥古斯丁、阿奎那、加尔文等伟大神学家都是逻辑的大师，他们的著作展示了严密的逻辑结构。

### 成全：逻辑的终极

在新天新地中：
- 我们将"面对面"认识上帝，不再需要间接推理
- 我们的思想将完全更新，不再有逻辑谬误
- 我们将完全理解上帝创造的理性结构

## 实践应用

### 在释经学中的应用

1. **识别论证结构**：
   - 保罗在罗马书中使用严密的逻辑论证
   - 约翰在约翰福音中使用"如果...则..."的结构

2. **检查神学一致性**：
   - 用逻辑检验教义是否与圣经整体教导一致

### 在护教学中的应用

1. **本体论论证**（安瑟伦）：
   - 用形式逻辑论证上帝的存在

2. **先验论证**（范泰尔）：
   - 论证逻辑本身预设了上帝的存在

## 结语

形式逻辑是上帝赐给人类的宝贵工具，它帮助我们：
- 清晰地思考
- 准确地表达
- 严密地论证

但我们必须记住：逻辑是工具，不是主人。我们的思想最终要降服于基督（哥林多后书 10:5），因为祂才是"道"的本体，是一切逻辑的源头。
